# rtk (Rust Token Killer) — разбор роста, уроки для дистрибуции zond

Исследование 2026-07-13. Цель — понять, как `rtk-ai/rtk` набрал ~70k звёзд за
~6 месяцев, и что из этого применимо к launch/дистрибуции zond. Факты —
из первоисточников (GitHub API, Homebrew analytics, CHANGELOG, HN Algolia);
внешние маркеры (Trendshift/dev.to) — secondary, помечены.

## Что это за инструмент

CLI-прокси, переписывающий dev-команды так, чтобы срезать потребление токенов
LLM на 60–90% (их заявка). Единый Rust-бинарь, zero-deps. Прозрачный
hook-механизм: `git status` → `rtk git status`, 0 токенов оверхеда. Ставится
на 9+ агентов (Claude Code, kilocode, antigravity, hermes…). Прямой конкурент-
сосед по нише «tooling вокруг coding-агентов», как и zond.

## Числа

- Звёзды: ~70k за ~6 мес.
- Homebrew: 85 259 установок/год (все за 5,5 мес.), 67k/90d, 18,7k/30d — рост фронт-загружен в весну, сейчас высокое плато.
- Каналы: curl-скрипт + **homebrew/core** + deb/rpm. Cargo-канала нет: имя `rtk` на crates.io занято чужим «Rust Type Kit».
- Ядро: `aeppling` (475 коммитов), `pszymkowiak` (214) + release-please автоматика.

## Уроки для zond

1. **Рост был product-led, не PR-led. HN не сработал** (нет launch-треда, два поста по 2–3 очка). → не переоценивать Show HN как событие; это разовый импульс, не двигатель.
2. **Одно-числовой value hook (`rtk gain`).** С первого README инструмент показывает «сколько ты сэкономил». Осязаемая цифра удерживает. → у zond есть coverage/findings, но нет одной команды-«вау»: *«N drift-находок / X% honest-2xx за 20 минут»* одной строкой. Кандидат в задачу.
3. **homebrew/core, а не только tap.** Попадание в дефолтный канal macOS-дева дало органический tail (~18k/мес). → ARV-387 (свой tap) — слабее; core-формула = следующая планка, когда наберём base для критериев homebrew (notability).
4. **Native-интеграция в агентов дала перелом (v0.30).** Skills/commands внутри Claude Code, потом 9 агентов. → у zond skills уже есть; расширение за пределы Claude Code (Cursor/kilocode/antigravity recall) — рычаг охвата.
5. **Нулевое трение установки + прозрачный hook.** 0 токенов оверхеда, ничего не надо учить. → держать `curl | sh` и npm тривиальными; не растить onboarding-трение.
