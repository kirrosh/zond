## Description

Уточнение роли MCP-слоя по итогам live-сессии: реальный pipeline прошёл через CLI + skill без вызовов MCP-tools. Направление B — сократить MCP-tools до тех, что не имеют CLI-эквивалента (structured JSON для diagnose/compare/coverage), оставить resources (workflow/правила). Пока меняем только документацию — реальный код MCP не трогаем до следующей живой сессии с включённым `.mcp.json`.
